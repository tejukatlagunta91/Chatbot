# 🧠 Mental Health Support Chatbot

A web-based **Mental Health Support Chatbot** built using **Streamlit** and **Google Gemini AI**. The chatbot provides empathetic conversations, emotional support, and general coping strategies while encouraging users to seek professional help when needed.

> ⚠️ **Disclaimer:** This application is **not a replacement for professional mental health care**.

## 📌 Overview

The Mental Health Support Chatbot is designed to:
- Listen empathetically to users
- Provide emotionally supportive responses
- Suggest general coping strategies
- Detect crisis-related language and recommend emergency resources

## ✨ Features

- Interactive chat interface using Streamlit
- AI-powered responses using Google Gemini 1.5 Flash
- Emotion-focused and compassionate replies
- Crisis keyword detection
- Safety-filtered AI responses
- Session-based chat history

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Generative AI (Gemini)

## 🚀 Quick Start

```bash
pip install streamlit google-generativeai
streamlit run mental_health_chatbot.py
```
